# letsdefend-soc-blue-team-investigations

Hands-on SOC analyst investigation labs focused on phishing analysis, endpoint monitoring, malware triage, and web attack investigations using Splunk, Wazuh, Wireshark, Linux, and MITRE ATT&CK mapping.

---

# Overview

This repository contains structured SOC investigation notes and recreated analyst workflows based on practical cybersecurity labs and blue-team exercises.

The investigations focus on:

* phishing detection and triage
* endpoint monitoring and alert analysis
* suspicious PowerShell activity
* web traffic inspection
* IOC extraction
* MITRE ATT&CK mapping
* detection engineering fundamentals
